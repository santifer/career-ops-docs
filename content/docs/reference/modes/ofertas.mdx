## What it does

ofertas reads multiple JD evaluations (already produced by oferta or auto-pipeline) and ranks them against each other on the six dimensions plus a normalized decision matrix. The output is a side-by-side comparison report and a ranked recommendation.

## When to use it

Use ofertas when you reach late-stage interviews for multiple roles and need a structured way to decide between them. Particularly useful for staff-level offers where comp packages and growth paths differ substantially.

## Example

```
After running oferta against three different offers, invoke `/career-ops ofertas` and reference the three evaluation files. The mode produces a ranked decision matrix.
```

## Gotchas

ofertas does not re-evaluate the offers; it ranks the existing reports. Make sure each offer has been scored individually first.
